# 📝 Notes App (React)

A simple Notes App built with **React.js** to add and delete notes dynamically with a responsive UI.

---

##  Features

* Add notes (heading + details)
* Delete notes
* Real-time UI updates
* Responsive layout


##  How It Works

* Notes are stored in state using `useState`
* New notes are added using array copy + `push`
* Notes are rendered using `map()`
* Delete uses `splice()` with index

---

## ▶️ Run Locally

```bash
git clone https://github.com/Akshit-bhardwaj/react-journey.git
cd notes-app
npm install
npm run dev
```
