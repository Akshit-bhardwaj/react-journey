# Local Storage in React

It demonstrates the use of **Local Storage in React** to store and manage data in the browser.

---

##  Features

* Store data using `localStorage.setItem()`
* Retrieve data using `localStorage.getItem()`
* Remove data with `localStorage.removeItem()`
* Clear all data using `localStorage.clear()`
* Handle objects/arrays using `JSON.stringify()` and `JSON.parse()`

---

##  Key Concept

Local Storage only stores **strings**, so complex data types must be converted before storing.

---

##  Use Cases

* Notes / Todo Apps
* User Data Storage
* UI Preferences

---

##  Tech Stack

* React.js
* JavaScript
* Local Storage API
