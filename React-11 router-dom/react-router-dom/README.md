Company Website (React + Tailwind)

A simple Single Page Application (SPA) built using React, React Router, and Tailwind CSS. This project demonstrates basic routing and a clean, responsive UI for a company website.

🚀 Features
⚛️ Built with React (functional components)
🧭 Client-side routing using React Router
🎨 Styled with Tailwind CSS
📱 Fully responsive design
🧩 Modular and scalable structure

📁 Project Structure
```
src/
│
├── components/     # Reusable UI components (Navbar, Footer, etc.)
├── pages/          # Page-level components (Home, About, Services, Contact)
├── App.js          # Main app with routing configuration
├── index.js        # Entry point
```

Tech Stack
React
React Router DOM
Tailwind CSS
JavaScript (ES6+)

 # What I Learned

Setting up routing in React using React Router.
Creating a SPA with multiple pages.
Structuring a React project into components and pages.
Styling using Tailwind CSS.
Building reusable UI components.
▶ How to Run
# Install dependencies
npm install

# Start development server
npm start
