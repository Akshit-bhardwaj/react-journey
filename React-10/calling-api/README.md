# React Axios API Call

##  Description
This project demonstrates how to fetch data from an API in React using Axios and display it in the UI.

##  Tech Stack
- React.js  
- Axios  
- JavaScript  

##  Features
- API call using Axios  
- Data fetching on component load (`useEffect`)  
- State management using `useState`  
- Display data in UI
